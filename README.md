## Installation

Build Cryptex from source and install dependencies:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/katniis/Cryptex
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Cryptex
   ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
